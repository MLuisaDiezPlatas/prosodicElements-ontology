Metaplasmc back to ToC or Class ToC

IRI: http://postdata.linhd.uned.es/ontology/postdata-prosodicElements#Metaplasm

A figure of transformation or metaplasm is a figure of diction that consists of altering the writing or pronunciation of words without altering their meaning. Some examples are dieresis, syneresis, sinalefa or hiatus. There is a pdprosodic:typeOfMetaplasm property whose range of values is a controlled metaplasm type vocabulary, which allows to specify the nature of it.
Is defined by
http://postdata.linhd.uned.es/ontology/postdata-prosodicElements
is in domain of
affects First op, success In op, type Of Metaplasm op

clausulaop back to ToC or Object Property ToC

IRI: http://postdata.linhd.uned.es/ontology/postdata-prosodicElements#clausula

The categorisation of the item according to the rhythmic ending of a line determined by the number of unstressed syllables following the last stressed one. This property is both valid for accentual and syllabic versification, so possible values include: masculine (zero syllables), femenine (one), dactylic, proparoxitone, etc.
Is defined by
http://postdata.linhd.uned.es/ontology/postdata-prosodicElements
has domain
Foot c
has range
